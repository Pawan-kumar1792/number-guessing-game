# Number Guessing Game 🎯

A simple and interactive Number Guessing Game built using HTML, CSS, and JavaScript where the player has to guess a random number between **1 and 100**.

The game provides instant feedback like:

* Higher number please ↑
* Lower number please ↓
* Congratulations message after correct guess 🎉

It also tracks:

* Total number of guesses
* Previous guess history
* New Game reset option

This project is beginner-friendly and helps in understanding basic JavaScript concepts like:

* DOM Manipulation
* Event Handling
* Random Number Generation
* Conditional Statements
* Functions
* User Input Validation

# 🚀 Features

## ✅ Random Number Generation

The game automatically generates a random number between **1 to 100** every time a new game starts.

---

## ✅ Smart Hint System

If the guessed number is:

* Greater than target → Shows **Lower number please ↓**
* Smaller than target → Shows **Higher number please ↑**
* Equal to target → Shows **Congratulations 🎉**

---

## ✅ Guess Counter

Shows how many attempts the player has taken.

Example:

Guesses: 7

---

## ✅ Guess History

Displays all previous guessed numbers in sequence.

Example:

1. 45
2. 72
3. 61
4. 58

---

## ✅ New Game Button

Allows the player to restart the game instantly with a new random number.

---

## ✅ Responsive UI Design

Works smoothly on:

* Laptop 💻
* Desktop 🖥
* Mobile 📱
